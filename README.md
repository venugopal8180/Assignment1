# Assignment1
This repository contains two simple Python programs:
Task 1: A Basic Calculator Program
Task 2: A User Greeting Program

These programs demonstrate basic Python concepts like:
1.Taking user input
2.Type conversion
3.Arithmetic operations
4.String concatenation
5.Printing output

Task 1: Basic Calculator Program
Description:
This program takes two numbers as input from the user and performs the following arithmetic operations:
Addition
Subtraction
Multiplication
Division

Concepts Used are:
input() function
int() type conversion
Arithmetic operators (+, -, *, /)
print() function

Working flow of task 1 program:
The user enters two numbers.
The program calculates:
Sum
Difference
Product
Quotient

The results are displayed on the screen.
If the second number is 0, division will cause an error (ZeroDivisionError).

Task 2: User Greeting Program
Description:

This program asks the user to enter their first name and last name, then displays a personalized greeting message.

Concepts Used are:

1.input() function
2.String concatenation
3.Printing formatted messages

working flow of task2 program:

The user enters their first name.
The user enters their last name.
The program combines both names.
A welcome message is displayed.
